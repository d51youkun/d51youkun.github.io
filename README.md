# BlueChat

LINE風のチャットアプリ（HTML / CSS / JavaScript）

## 使い方

1. `index.html` をブラウザで開く
2. 名前を入力して「はじめる」
3. 友だち追加 → パスワードを発行 / 入力
4. トーク・グループ作成でメッセージのやり取り
5. マイページでプロフィール画像をアップロード

## 注意

- データはブラウザの localStorage に保存されます（このデバイスのみ）
- 他のデバイスへの引き継ぎはできません
- 友だち追加のパスワードは、同じブラウザに登録されたユーザー間で有効です

## ファイル構成

- `index.html` — アプリ本体
- `styles.css` — スタイル（青ベース）
- `app.js` — ロジック
